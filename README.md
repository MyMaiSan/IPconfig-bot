## ⚡ Использование скрипта

1. Выполняем ```https://github.com/MyMaiSan/IPconfig-bot```
3. Вставляем TOKEN бота в ```config.py```
4. Выполняем ```pip install -r requirements.txt```
5. Запускаем ```bot.py```
